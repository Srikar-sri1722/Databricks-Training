# Week 1 - Day 2

## Objective
To practice and strengthen SQL concepts by performing different types of queries on the `Employees` table using filtering, grouping, sorting, logical operators, and pattern matching techniques.

---

## Database Table Used
### Employees Table
The project uses a single table named `Employees` containing employee details such as:
- Employee ID
- Employee Name
- Department
- Salary
- City
- Experience

---

## Tasks Completed
- Created the `Employees` table
- Inserted sample employee records
- Executed 55 SQL practice queries
- Performed data filtering using `WHERE`
- Used aggregate functions with `GROUP BY`
- Applied conditions using `HAVING`
- Retrieved top records using `ORDER BY` and `LIMIT`
- Practiced:
  - `DISTINCT`
  - `IN`
  - `NOT IN`
  - `BETWEEN`
  - `LIKE`
  - Logical operators (`AND`, `OR`, `NOT`)
- Generated query outputs in tabular format
- Documented all query results in GitHub markdown format
